# New Feature Proposals for DHConsole

This document outlines potential new features for DHConsole based on available server commands that are not yet implemented in the UI.

## 1. Simulated Universe (Rogue) Tools

### Commands
*   `/rogue money <amount>`
*   `/rogue buff <buffId>`
*   `/rogue miracle <miracleId>`
*   `/rogue roll`
*   `/rogue enhance`

## 2. Currency Wars (Grid Fight) Tools

### Commands
*   `/grid gold <amount>`
*   `/grid role <roleId> <tier>`
*   `/grid equip <equipmentId>`
*   `/grid orb <orbId>`
*   `/grid consumable <consumableId>`
